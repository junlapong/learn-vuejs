# Learn Vuejs

## Setup

[nvm](https://github.com/nvm-sh/nvm) and node LST version

	curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
	nvm install --lts


### Install Vue CLI

	npm install -g @vue/cli


### Create New App

	vue create <app-name>
